以下是一些在日常 SFDC 工作中非常有帮助的网站和工具。如果你也在使用某些工具，欢迎在评论区分享。如果我发现了更多好用的内容，我会持续更新这篇文章。

  • 网站：

    • https://developer.salesforce.com/

    • https://help.salesforce.com

    • Trailhead： https://trailhead.salesforce.com/

    • SFDC 工具集： https://cloudtoolkit.co/

    • Click to Create LWC 设计工具： https://www.lightningdesignsystem.com/guidelines/builder/click/

    • Unofficial SF： https://unofficialsf.com/

    • 正则表达式生成工具： https://regex-generator.olafneumann.org/?sampleText=2020-03-12T13%3A34%3A56.123Z%20INFO%20%20%5Borg.example.Class%5D%3A%20This%20is%20a%20%23simple%20%23logline%20containing%20a%20%27value%27.&flags=i&onlyPatterns=false&matchWholeLine=true&selection=

    • 批量更新多个配置文件： https://profilepowertools.com

    • SFDC 精选资源大全： https://github.com/mailtoharshit/awesome-salesforce

  • 浏览器扩展：

    • SFDC Advanced Code Searcher：在整个 org 中搜索所有代码
    • Color Favicons：区分沙盒和生产环境，避免在两个环境之间误操作（哈哈）
    • Inspector：查看所有数据，通过查询导出数据
    • Mass Editor：在列表视图中批量编辑
    • Show API Name：无需介绍，顾名思义
