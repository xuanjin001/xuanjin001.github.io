Personal Accounting -- Receipt Tracker

March 22, 2026 4-minute read
Xuan Jin

I have been trying to find an app where it can record my day to day purchases, so that I can compare the price from costco vs target, which can help me evaluate how to spend my money. I tried with mint, it only has a really generic spending structure, and the total amount that you spent from each merchandise, and depends on where you are buying things, it will generically tell you how you are spending your money. However, I want it to be more specific.

Back in 2020, that’s when I first started on budgeting, I did the whole budgeting thing when I got my very first job in 2016 to 2018, back then I used excel and I will manually typing in how much I spend in FoodMaxx, winco, etc. In that way, I can see how i can properly pay off my credit card and also manage all other costs. I didnt do line by line booking back then, only because it is too much work. LOL. Then in 2020, with pandemic, I feel like I need to do budgeting again so that in that time, I can make sure I truly only spend money on the things I need. That kind of tool doesn’t really exist. Or, you have to pay a lot to get an app, and those app still require a lot manual entries and setup. You can connect things with bank for those apps, like quicken simplifi, rocket money, etc, those app/services are sometimes good at one area, not the whole thing.

I want to have an app which can record line by line costs, and eventually can match the costs to what shows up on my credit card bill. I couldn’t find one that fits my needs, so I started manually doing it, organize receipts and checking my bank accounts, etc. It was a lot of manual work, 2 receipts, it will take me about 40 minutes, I need to draw lines on a notebook and do the math to add everything up. I think I did that for 3 weeks, and then just stopped. During pandemic, there are too much going on, and I was staying indoors a lot, with anxiety and everything, I don’t find doing budgets relaxing at the time. I was thinking, hey, were there a better way to do this? What if I take a picture of the receipt and have a program to translate it?? Then I found tesseract OCR to extract from photo, I tried with couple receipt, but it still would miss some of the lines, which is a bit no no, cause that means, I need to manually add the lines, and the correction effort is actually quite bit, not only sometimes it missed couple lines from receipt, it also lined up the item and cost off, meaning, it would line first item with the price for the second item on the receipt, which is something we don’t want at all. At the time, that’s what I was working with, sometimes, some mom-and-pop kind of receipt doesn’t even work, so I was doing those manually. Soon, I give up again. LOL.

Now, finally, we have AI!!!!!

I introduce you “Receipt Tracker”! This is in beta still, and there are a lot of things I still need to refine, but I am pretty happy with the current version of it!

Dashboard page:

When you click on “Add receipt”, it will direct user to the Dashboard page, and user would have the options to “Upload File”, “Camera”, “Paste Text”, “Manual Entry”. Each tab/function is pretty obvious, once anything is uploaded, I am using LLM api to channel the AI LLM somewhere in the universe lol, and have LLM to help process the data/image I uploaded, and match with the columns.

Once saved, it will have everything listed in transaction.

This program can also do data analysis as well.

There are still couple improvements I need to do with this, but this version is great! With this, it can help with budgeting later on, and also with tax, those are all itemized for each transaction.

For this project, I used Gemini’s models. Gemini’s model can not only read the image or text and translate them into database compatible data; Gemini can also process data for analysis purposes.
