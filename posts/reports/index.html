Reports

Definition: Reports are lists or summaries that allow you to aggregate and analyze your data in different ways.

Standard Reports:

` `

  • Build in

` `

  • Stored in standard folders

` `

  • Can be customized

` `

  • Saved as custom reports since they can’t be overwritten

Custom Reports:

` `

  • Can be built from scratch

` `

  • Can be created by customizing standard reports

` `

  • Must be saved in a custom, personal, or unfiled folder

Finding reports on the Enhanced Reports Tab

Report & Dashboard Folder Sharing: Traditional Model Access to reports and dashboards is controlled through folders on the Reports tab

Traditionally, all users with access to the folder have the same level of access

` `

  • Access can be Ready-Only or Read/Write

` `

  • Folders can be accessible to all users, hidden form all users, or shared using public groups or rules

Access to folders does not roll up in the role hierarchy

Report and Dashboard Folder Sharing: Enhanced Model

Enhanced folder sharing gives users finer control over how they share their reports and dashboards.

Different groups of users can be given different access to the same folder

Alert: Enabling enhanced folder sharing may change the existing access to reports & dashboards, depending on their user permissions

Using report options for quick customizations:

` `

  • Always available at the top of every report

` `

  • Options depend on the type of data being reported on

` `

  • Make changes in a few clicks, then save a new report

Report Format: Summary Report

` `

  • Records sorted into groups with subtotals

Matrix Report

` `

  • Summarizes data in a grid against horizontal and vertical criteria

` `

  • Provides totals for both rows & columns

Grouping by Date Fields

` `

  • While grouping a report by a date field, select “Group Dates By” from the group menu to specify the grouping time frame

Report Filters and Filter Logic:

Add up to 20 additional filters to a report, directly in the Filters pane, using the Add button or by dragging in fields from the Preview pane. In addition, use filter logic to:

` `

  • Link multiple filters using “AND”, “OR”, “NOT” operators to create sophisticated conditions

` `

  • Offset priority conditions using parentheses and nested parentheses

Reports & Dashboards Continue

` `

  • Create dynamic date reanges fro reports using relative date values in filters

` `

  • Extra flexibility: can use variables in relative date values for days, quarters, and years

Adding a Custom Summary Formula to a report

` `

  • Up to five per report

` `

  • Cannot reference other summary formulas

` `

  • Can reference formula fileds in the data

` `

  • Only display on summary rows, not detail(record) rows

Use Highlight exceptional values in key fields to make them stand out

` `

  • Conditional highlighting

Use Chart to visually present the data in your summary rows

` `

  • Bar Charts

    ` `

    • Each bar represents data from a grouping

    ` `

    • Use horizontal to compare many groups

    ` `

    • Use vertical to compare fewer groups or when grouping by a date field

` `

  • Line Chart

    ` `

    • Each point represents data from a grouping

    ` `

    • Use when grouping by a date field

    ` `

    • Great for displaying changes over time

    ` `

` `

  • Pie/Donut Chart

    ` `

    • Each wedge represents data from a grouping

    ` `

    • Size of wedge is determined by summary field

    ` `

    • Use to compare Shares of the grand total

    ` `

    • Use a donut chart to display the ground total

    ` `

` `

  • Funnel

    ` `

    • Each segment represents data from a grouping

    ` `

    • The size of segments is determined by the summary field of each grouping

    ` `

    • Use funnel charts to compare the summarized values of groupings

` `

  • Scatter Chatter

    ` `

    • Plot 2 summary fields against each other

    ` `

    • Visualize the correction between values in a group

    ` `

    • Enable the report builder upgrade to use Scatter Charts

` `

  • Chat Types: Multiple Grouping

    ` `

    • Side-by-side bar:

      ` `

      • Use this type of chart for multiple groupings

      ` `

      • Each source is represented by a set of bars

      ` `

    ` `

    • Stacked Bar

      ` `

      • Use this bar graph to display the proportions between values

      ` `

      • It displays a single bar for each primary value, with secondary values shown on the same bar

    ` `

    • Stacked to 100% Bar

      ` `

      • Use this for proportions between values in each grouping

      ` `

      • All bars are the same overall length

      ` `

    ` `

    • Grouped Line

      ` `

      • Each bar represents an ordered set of data

      ` `

      • Missing values are displayed as gaps

    ` `
