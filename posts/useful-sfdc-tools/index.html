Useful SFDC Tools

July 29, 2021 One-minute read

Developer

There are some sites and items that’s really helpful on our daily basis while working in SFDC. If you have some tools that you are using, please post that in the comments as well. I will continue update this post if I found more items.

  • Websites: 

    • https://developer.salesforce.com/

    • https://help.salesforce.com

    • Trailhead: https://trailhead.salesforce.com/

    • Toolkit for SFDC: https://cloudtoolkit.co/

    • Click to Create LWC designing tool: https://www.lightningdesignsystem.com/guidelines/builder/click/

    • Unofficial SF: https://unofficialsf.com/

    • Regex translator: https://regex-generator.olafneumann.org/?sampleText=2020-03-12T13%3A34%3A56.123Z%20INFO%20%20%5Borg.example.Class%5D%3A%20This%20is%20a%20%23simple%20%23logline%20containing%20a%20%27value%27.&flags=i&onlyPatterns=false&matchWholeLine=true&selection=

    • Mass Update on mutilple profiles: https://profilepowertools.com

    • A curated list with all things SFDC: https://github.com/mailtoharshit/awesome-salesforce

  • Extensions: 

    • SFDC advanced code searcher: search through all of the code that’s in your org
    • Color favicons: identify sandbox or prod, so you don’t accidentally screw up between the environment LOL
    • Inspector: see all data, export data using query 
    • Mass Editor: mass edit when on list view 
    • Show api name: no need introduction
